# movie-library
A Python tool to build your personal movie database and visualize your favorite genres.

### What it does
- **Collects Data:** Input your favorite movies, their genres, and ratings.
- **Filters Favorites:** Automatically identifies and lists top-rated movies (Rating 9+).
- **Genre Analysis:** Generates a professional pie chart showing your movie taste distribution.

### Tech Stack
- **Python**
- **Pandas** (Data Processing)
- **Matplotlib** (Data Visualization)

### Quick Start
1. Install dependencies: `pip install pandas matplotlib`
2. Run: `python movie_library.py`
